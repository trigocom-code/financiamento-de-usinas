● Monitor regulatório SOA · regeneração programada

O mercado elétrico se auto-regula diariamente. Estes materiais acompanham a evolução: cada cartão consulta o repositório no carregamento e mostra a hora exata da última regeneração e um selo de frescor.

Verde = atualizado nas últimas 24h, pode compartilhar à vontade. Amarelo = 1 a 7 dias, ainda confiável mas confira antes de cravar dado. Vermelho = mais de 7 dias, NÃO compartilhe — material defasado, peça regeneração ao monitor.

Como o frescor é medido. Cada cartão faz uma consulta à API pública do GitHub (`api.github.com/repos/.../commits?path=…`) no momento em que você abre este catálogo. O timestamp exibido é o do último commit que tocou aquele arquivo no repositório — quando o monitor regulatório SOA (digest diário + regeneração agendada) o reescreveu pela última vez. O carimbo no canto superior direito mostra a hora EM QUE VOCÊ está vendo este catálogo.
